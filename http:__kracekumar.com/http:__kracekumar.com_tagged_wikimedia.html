First Wikimedia DevCamp Experience

9-11th Nov, 2012 was wikimedia dev camp in Bangalore. This was my first wikimedia dev camp. Dev Camp was attended by 60+ people who were wikimedia developers, contributors and first timers. The main focus was Language Engineering, Mobile Development, User Interface & Experience.

As a result of lot quite useful applications/work came out. Here is list of applications

Demos

  2. Variants, WebFonts and ULS on mobile - Brion

    https://github.com/brion/Wikipedia/commits/uls

    https://github.com/brion/Wikipedia/commits/variants

    https://github.com/brion/Wikipedia/commits/fonts

  3. WikipediaMobile for FirefoxOS - Patrick

    https://github.com/wikimedia/WikipediaMobileFirefoxOS
