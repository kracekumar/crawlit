Download youtube videos without using any addon or software

This Hack is applicable in Ubuntu,but the logic is same for all OS. . .
All the flash videos which is displayed in the browser gets streamed from the server and saved in your hard disk and rendered back to the browser.Once we find out the destination of the videos we can just copy and paste the videos in the desired folder…Main point to remember here is once the URL containing the video is closed the no longer the video file gets saved in the computer and all the previously downloaded will be deleted… .(Tech term: UDP connection is closed)

Open the website containing video.
1.Start playing the video.
2.Now pause the video(don’t worry still video will be getting downloaded).
3.Go to /tmp folder and you see video file with Fl*****.
4.Now copy paste the video in your favorite destination.

It will long time to download the entire video depending on your video file size… .

Web ripping using wget

You can have entire copy of the website(offline browser) using wget.Web ripping normally allows the user to browse the ripped site as if their

online.Their are lots packages available under GNU GPL license with fully loaded GUI features such as HTtrack ,PageNest etc…

wget is used in recursive mode yields the desired result.

Command:

kracekumar@kracekumar-laptop:wget -r  www.ripthesite.com

All the pages which can be visited by the web robots can be ripped,if you are unable to rip a page then you to take a look at the permission(robots

permission)

www.ripthesite.com/robots.txt file will yield the complete details of web robots permission.

now go to the current folder and view the mirrored site… PWD will yield the present directory name
